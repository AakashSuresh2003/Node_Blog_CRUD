Blog Backend

This project demonstrates the functionality of various routes:

Authentication Routes

  • POST /register: Register a new user.
  • POST /login: Log in with existing credentials.
  • GET /logout: Log out the current user.
  • GET /refetch: Refetch user data.

Blog Routes

  • GET /blogs: Retrieve all blogs.
  • GET /blogs/:id: Retrieve a specific blog by ID.
  • POST /blogs: Create a new blog.
  • PUT /blogs/:id: Update an existing blog by ID.
  • DELETE /blogs/:id: Delete a blog by ID.

User Routes

  • GET /users: Retrieve all users.

For more information, please refer to the project's GitHub repository.
